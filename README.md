# panda-clone-auth

Help me engineer the clone of this web application in express, react, mongoose, mongodb. The name of the web application is pandaloc and it is hosted at pandaloc.com. Start by designing and implementing an authentication solution with added features: phone number verification, reset password for both Owners and Renters

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/panda-clone-auth.git
cd panda-clone-auth
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
